How to Run This Locally

Open PowerShell, then cd into the repo root folder before running any command. The repo root is the folder that contains index.html, data/, and scripts/.

Run commands one at a time. Replace the SEC user-agent value with a real contact string before using network-enabled SEC commands.

SEC USER-AGENT

The SEC user-agent is not an account and it does not email you. It identifies the script politely to the SEC. Example: "Taylor infusednft@gmail.com".

Pipeline Status

The current path moves from source lookup and cache planning into review-only candidate files. Production validation, review, and promotion are future phases.

SEC lookup/cache -> filing plan -> filing fetch -> filing inspect -> signal extraction -> signal report -> candidate preview -> candidate file writer -> future validation/review/promotion

Recommended One-Command Workflow

Run this from PowerShell in the repo root. The browser only displays and copies commands.

Replace the example SEC user-agent with your real contact string.
1. Dry-run and preview from local cache 
python scripts/sec_pipeline_run.py --ticker AAPL --forms 10-K,10-Q,8-K --limit 10
2. Network-enabled cache and filing fetch 
python scripts/sec_pipeline_run.py --ticker AAPL --forms 10-K,10-Q,8-K --limit 10 --allow-network --user-agent "Taylor infusednft@gmail.com"
3. Write review-only candidate file 
python scripts/sec_pipeline_run.py --ticker AAPL --forms 10-K,10-Q,8-K --limit 10 --allow-network --user-agent "Taylor infusednft@gmail.com" --write-candidates --force
Advanced/manual workflow summary
Use individual scripts only when inspecting an intermediate artifact. 
python scripts/ingest_candidates.py --candidates data/candidates/cik_mappings.json --candidate-kind cik-mappings --summary-only
python scripts/sec_fetch_cache.py --ticker AAPL --user-agent "Taylor infusednft@gmail.com" --dry-run
python scripts/sec_submissions_inspect.py --cache-file data/cache/sec/submissions_CIK0000320193.json --forms 10-K,10-Q,8-K --limit 10
python scripts/sec_filing_plan.py --cache-file data/cache/sec/submissions_CIK0000320193.json --forms 10-K,10-Q,8-K --limit 10 --json
